# Connexus — AI Context

Premium web studio for Czech financial advisors. Solo founder: Jakub Barák.

## Tech stack
- Astro 4.x (static generation)
- Tailwind CSS v3
- React islands where needed
- Cloudflare Pages deployment
- Formspree for form handling
- GA4 for analytics
